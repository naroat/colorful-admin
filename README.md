## 项目介绍

演示：`admin.colorful.ranblogs.com`

目录：
```
api: 后端代码
admin: 后台前端代码
```

### 版本信息
```
php: ">=7.3"

hyperf: "~2.2.0"

vue-element-admin: "4.4.0"
```

### 功能概述

- Jwt认证登录
- 修改密码
- 文件上传(支持本地和阿里云oss)
- 管理员管理
- 菜单管理
- 权限管理
- 角色管理
- 网站设置
- 文章管理
- 管理员操作日志
